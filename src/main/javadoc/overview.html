GML Conformance Test Suite - Documents

This executable test suite (ETS) verifies the conformance of the instance under test (IUT) with respect to all relevant specifications.

The test run arguments are summarized in the following table. The Obligation descriptor can have the following values: M (mandatory), O (optional), or C (conditional).

Test run arguments
Name Value domain Obligation Description
iut URI M A URI referring to the instance document under test. Ampersand ('&') characters must be percent-encoded as '%26'.
sch URI O A URI referring to a Schematron schema that defines supplementary constraints.
